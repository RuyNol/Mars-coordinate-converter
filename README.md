# GIS坐标转换工具
点击下载：https://github.com/RuyNol/Mars-coordinate-converter/releases

## 功能特点
1. JSON数据坐标转换
2. 文件批量转换
3. 支持多种坐标系：GCJ-02、BD-09、WGS84、CGCS2000
4. 支持多种输出格式：CSV、Shapefile

## 使用方法
1. 直接运行 `GIS坐标转换工具.exe`
2. 无需安装Python或其他依赖

## 系统要求
- Windows 7/8/10/11
- 无需管理员权限

## 文件说明
- `GIS坐标转换工具.exe` - 主程序
- `logo.ico` - 程序图标
- `qrcode.jpg` - 二维码图片

## 注意事项
1. 程序运行时会在临时目录解压必要文件
2. 确保有足够的磁盘空间
3. 部分功能需要GDAL支持
